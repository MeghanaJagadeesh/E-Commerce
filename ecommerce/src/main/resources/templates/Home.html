About

Our ecommerce shopping application is a result of dedication, collaboration, and technical expertise. Developed as part of implementing our skills, this platform showcases the implementation of Spring Boot and Thymeleaf in creating a functional and efficient online shopping experience.

Project Objective

The primary objective of this project is to demonstrate our understanding and application of key concepts in software development. By utilizing Spring Boot's robust framework and Thymeleaf's templating engine, we aimed to build an intuitive and responsive ecommerce platform.

Technologies

Java 16, Spring Boot, Thymeleaf template engine, Hibernate with JPA, MySQL, HTML, CSS, Javascript
